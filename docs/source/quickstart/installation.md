# Installation

## Latest Version
**Install from PyPI**
```
pip install fastface -U
```

**Install from source**
```
git clone https://github.com/borhanMorphy/light-face-detection.git
cd light-face-detection
pip install .
```

## Specific Version
**Install from PyPI**
```
pip install fastface==x.x.x
```

**Install from source**
```
git clone https://github.com/borhanMorphy/light-face-detection.git
cd light-face-detection
git checkout vx.x.x
pip install .
```
